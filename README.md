# Expense Tracker (Java)

A console-based expense tracking application built using Java.  
The project focuses on core programming concepts such as object-oriented design, collections, and file handling.

## Features
- Add expenses with category, amount, and date
- View all recorded expenses
- Delete expenses
- Persistent storage using file handling
- Menu-driven console interface

## Tech Stack
- Java
- Object-Oriented Programming (OOP)
- File I/O
- Collections (ArrayList)
